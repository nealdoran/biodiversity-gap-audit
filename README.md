# Biodiversity Data Gap Audit

Interactive dashboard cross-auditing IUCN Red List threatened species 
classifications against GBIF occurrence records.

**Live app:** [link coming]

## What This Shows
61.8% of Critically Endangered species have fewer than 10 documented 
occurrence records in GBIF. This app makes that data gap visible and 
explorable across taxonomic classes and threat categories.

## Data Sources
- IUCN Red List (50,576 species)
- GBIF (1.9M species, 26M occurrence records)
- Database: bio_database_v2.db
